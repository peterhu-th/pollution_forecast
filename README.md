# 空气质量协同变化预测与分级决策模型

## 0. Question

在评价空气质量时，PM2.5、PM10、O₃、NO₂、SO₂、CO是主要的污染物，其浓度受污染源排放和气象条件共同影响。比如，温度与湿度就会显著改变污染物浓度的时空分布。这些影响可能会立刻反映为污染物浓度变化，也可能会有一定滞后性，即此前一段时间的温度、湿度导致了后来的污染物浓度变化。
如果能够找出温度、湿度对污染物浓度的影响，就可以更好地了解空气质量变化规律、更有效地预警空气污染，保障人民群众身心健康。
附件中给出了北京房山良乡地区2026年1月1日-3月31日逐小时的空气质量监测数据，包含PM2.5 (μg/m³)、PM10 (μg/m³)、O₃ (μg/m³)、NO₂ (μg/m³)、SO₂ (μg/m³)、CO (mg/m³)和同时段温度（℃）、相对湿度（%）。请根据所给数据，如果需要可以结合其它数据(自己决定要不要补充数据、补充什么数据、如何补充)，建立模型回答如下问题：

1、分析单一污染物浓度与时间、温度、湿度之间的关系，并给出合理的解释。
2、建立数学模型，基于时间、温度、湿度预测六种污染物浓度的协同变化规律。
3、请根据温湿条件与污染物变化规律，建立空气质量分级、预测模型，帮助良乡校区同学更好地安排每天活动。

## 1. Overview

本仓库包含了用于分析并预测空气中六种主要污染物协同变化规律的深度学习代码，以及一款基于 **Streamlit** 构建的智能环境评价前端可视化引擎。

**主要方法与核心思路：**

- **特征挖掘阶段**：使用交叉相关函数 (CCF) 与 XGBoost 结合的 SHAP 值特征归因分析，衡量时间、温度、湿度等气象变量对单一污染物的即时或滞后影响。
- **时空预测机制**：构建了多输入多输出 (MIMO) 架构的深度时空预测网络（基于重构的 LSTM）。编码器高效捕捉历史气象与排放的序列时间依赖性，并输出未来多时间步的浓度走势。
- **动态评级与前端决策引擎**：为了解决问题三，引入了**模糊综合评价法 (FCE)**。我们开发了 `app.py` 交互式大屏，自动拉取历史 24 小时真实数据送入深度学习模型，预测未来 12 小时浓度，并在前端通过隶属度矩阵合成算法实时得出 V1~V4 的空气质量健康建议与智能校园活动指南。

## 4. Usage

复现实验并回答题目中三个问题的操作步骤如下：

### 环境准备

所有命令应当在一个独立的 Conda 环境中执行：

```bash
conda create -n pollution python=3.12
conda activate pollution
pip install -r requirements.txt
```

### 问题一：特征分析

1. 将原始监测数据放置入 `data/raw/` 文件夹中。
2. 运行 `notebooks/Data_Preprocessing.ipynb`，生成合并的数据集 `data_merged.csv`，以及 `train_set.csv`, `val_set.csv`, `test_set.csv` 等。
3. 运行 `notebooks/CCF_and_SHAP.ipynb`，得出温湿条件对各污染物的最大滞后阶数及 SHAP 特征归因图表。

### 问题二：协同预测规律构建

1. 按需调整 `src/config.py` 中的超参数。
2. 开始模型训练获取权重：
   ```bash
   python src/train.py
   ```
   训练完毕后最优模型将保存至 `results/models/best_forecaster.pth`。
3. 执行模型测试与评估：
   ```bash
   python src/test.py
   ```
   可计算各项评价指标并在终端打印 RMSE 与 MAE，同时输出可视化曲线。

### 问题三：智能 AQI 评级与动态决策 (Web 端)

为了直观回答第三问“如何帮助同学安排每天活动”，我们提供了所见即所得的可视化 Web 引擎：

1. 确保上一步的 `best_forecaster.pth` 与合并数据集存在。
2. 在终端执行以下命令拉起系统大屏：
   ```bash
   streamlit run app.py
   ```
3. **功能操作**：
   - 在左侧侧边栏，任意滑动调整时间刻度 $T$。
   - 引擎会自动截取 $T$ 时刻前 24 小时的真实气象信息进行推理。
   - **模块 A & B**：实时基于预测浓度的 PM2.5、PM10、O3 构造隶属度评价矩阵，敲定 V1 极度适宜 ~ V4 中重度污染的最终等级，并反馈健康与运动贴士。
   - **模块 C**：无缝呈现过去监测值到未来模型预测值的时间走势。
   - **模块 D**：内置白盒溯源模式，一键展开可查看矩阵的底层 $\LaTeX$ 数学相乘过程，为论文的严密性背书。
